## 해석

1. **표층 패턴 스멜은 수렴** — K6 약한의무(κ=.82)·K9 모호어(κ=.65)처럼 어미·사전으로 정의되는
   스멜은 규칙과 LLM이 거의 동일 판정. 규칙 기반의 강점 영역.
2. **의미 판단 스멜은 발산** — K2 불완전·K3 주체누락은 agreement가 58%로 낮다.
   LLM은 제목형·명사종결 행을 거의 전부 불완전/주체누락으로 판정(더 엄격),
   규칙은 조사·종결어미 단서가 있어야 검출. 유병률이 높아 kappa 역설(높은 일치에도 κ≈0) 발생 —
   4분할 원자료를 병기해 보고해야 함.
3. **비대칭 방향이 체계적** (McNemar p<.001):
   - LLM-우세 검출: K1 복합(41:0), K7 범위모호(41:1), K3 주체누락(123:4) — 의미 결합·범위 해석이 필요한 유형
   - 규칙-우세 검출: K11 미정의약어(63:13), K10 정량부재(22:2), K4 불완전조건(14:0) —
     문서 전역 스캔(약어 정의 존재 여부)이나 키워드 공기(共起) 판정이 필요한 유형.
     LLM은 문장 단위 프롬프트라 문서 맥락이 없어 약어를 관대하게 판정
4. **결론 방향**: 규칙과 LLM은 대체 관계가 아니라 **상보 관계** — 표층 스멜은 규칙,
   의미 스멜은 LLM, 전역 맥락 스멜은 규칙(문서 인덱스)이 우세. 하이브리드 파이프라인 근거.

## 상용 LLM 2종 확장 (GPT-4o 추가)

### 규칙 vs GPT-4o (n=300)

GPT-4o는 Claude보다 더 관대하게 검출한다 — 특히 K1 복합(GPT 146 vs 규칙 4), K7 범위모호(98 vs 17), K5 수동(66 vs 14).
any-smell: 규칙 91.3% vs GPT 99.7%. 규칙과의 상위 일치: K6 약한의무 κ=.45, K11 κ=.39, K8 κ=.39.

**핵심 발견 — 의미 스멜에서 LLM끼리는 수렴, 규칙과는 발산**:
K2 불완전 명세에서 두 상용 LLM 간 κ=.60(substantial)인데 각 LLM과 규칙 간에는 κ≈0.
즉 두 LLM은 규칙이 포착하지 못하는 **공유된 의미 해석 기준**으로 판정하고 있다.
이는 "의미 판단 스멜은 LLM 계층에 위임"이라는 하이브리드 설계의 독립 재현 증거다
(단일 LLM의 특이 성향이 아니라 상용 LLM 2종에서 일관).

반면 K6 약한의무는 규칙-Claude κ=.82 > LLM 간 κ=.47 — 표층 스멜은 여전히 규칙이 기준선이다.

## 방법 기록·한계

- Claude 판정: 대화형 세션 zero-shot, temperature 미고정 — API 기반 재실행으로 확정 권장
  (`llm_judge.py --backend anthropic`, 캐싱 지원). GPT-4o 판정은 API, temperature=0, 캐시 보존
- K4는 두 LLM 모두 ~0건 — 프롬프트의 K4 정의가 좁게 해석됨. 정의문 보강 필요 (공통 약점)
- 원문·표본 파일은 LOCAL_ONLY; 본 저장소에는 라벨(0/1)과 통계만 커밋
