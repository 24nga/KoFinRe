# 금융 도메인 vs 공공 다도메인 실험 결과 비교

> **KoFinRe v2.8 19종 smell taxonomy 기준 / 4 도메인 데이터셋 정량 비교**
> 한국어 요구사항 평가 루브릭의 도메인 무관(domain-agnostic) 일반화를 입증하기 위한 분석.

---

## 0. 한 줄 요약

**S5 주체누락은 4 데이터셋 모두 Top 3 안에 들어 한국어 SI 작성 관행이 도메인 무관함을 입증**. 단 **공공 다도메인(D5)에서만 Top 1로 부상한 S18 추적ID부재(69.7%)** 는 HWP 본문 평면 텍스트의 구조적 결함이며, 이는 **양식 정형성 효과(-82pp)가 도메인 변경 효과(+22pp)보다 3.7배 크다**는 사실로 정량 확인되었다.

## 5. 도메인 간 핵심 차이

| 차원 | 금융 도메인 | 공공 다도메인 |
|---|---|---|
| **공통 우세** (한국어 SOV·주어생략 영향) | S5 주체누락 (D1·D2·D4 모두 Top 3) | S5 주체누락 (Top 3) |
| **금융 특유** | **S4 약한의무 + S2 불완전 결합** (양식 명세서 영향) | (약함, 평면 RFP 본문이라 의무 표현 형식 결여) |
| **공공 특유** | (D4 외 양식 명세서 부재) | **S18 추적ID부재** 압도적 (HWP 본문 평면) |
| **양식 정형성 효과** | D2/D4 정형 → S3 모호어 ↓ (4~26%) | D5 XLSX 17.3% vs HWP 99~100% (편차 5배+) |
| **v2.7/v2.8 신규 smell 기여** | (해당 시점 미평가) | **S13 + S15 + S18 = D5 Top 5 중 3개** ⭐ |

## 6. "양식 정형성 > 도메인" — 결정적 발견

도메인 변경의 효과와 양식 정형성 변경의 효과를 동일한 단위(percentage point)로 비교:

| 비교 축 | Smell 비율 변화 |
|---|---:|
| 도메인 변경 (D1 금융 → D5 공공) | 65.3% → 87.5% = **+22.2 pp** |
| **양식 변경 (D5 내부, HWP 평면 → XLSX 정형)** | 99~100% → 17.3% = **-82 pp** |
| **상대비 (양식 효과 / 도메인 효과)** | — | **3.7배** |

→ **품질 핵심 인자는 도메인이 아니라 양식 정형성·ID·출처 추적**.
   다른 표현으로: "한국어 요구공학 품질 개선의 ROI 1순위는 양식 정형화."

## 8. 4가지 핵심 결론

1. **양적 검증의 안정성**: D5(4,075 req)가 금융 3 데이터셋 합(362 req)의 **11.3배** → 통계적 안정성 크게 향상. 도메인 사전 미보강 상태에서도 단일 패스로 평가 가능.

2. **S5 주체누락의 도메인 무관성**: **4/4 데이터셋에서 Top 3** → **한국어 SOV 어순·주어 생략 허용·평서형 종결**이 도메인을 가로질러 일관 관찰. 본 연구의 RQ4 답 ("한국 RFP 품질 이슈는 정제 수준이 아니라 언어·문화적 작성 관행")의 다도메인 강화 증거.

3. **공공 특유 S18 (추적ID부재 69.7%)의 발견**: 금융 데이터셋은 양식 명세서(D4 XLSX) 위주라 ID 자동 부여, 공공 RFP HWP 본문은 평면 텍스트라 추적 ID 부재. → **양식 자체가 가장 강한 품질 신호**.

4. **v2.7/v2.8 신규 smell의 정량 가치**: D5 Top 5 중 **S13 추측표현 + S15 지시어모호 + S18 추적ID부재 = 3종이 신규** (60% 점유). 기존 10종으로는 공공 RFP의 핵심 결함의 절반 이상을 보지 못함. **CMMI/ISO/INCOSE 표준 정렬의 정량 가치를 다도메인 데이터에서 입증**.

## 9. 한계 및 후속

| 한계 | 설명 |
|---|---|
| 비대칭 시점 평가 | D1·D2·D4 평가가 v2.0~v2.3 시점이라 S11~S19 데이터 부재. v2.8 재평가 필요 (D4는 로컬 환경에서) |
| D5 단일 데이터셋 | 다도메인이지만 1 패키지 (2013 발주 사례) — 2020~2026 최신 공공 RFP 비교 필요 |
| 평가자 gold 부재 | 자동 검출만으로 우세 패턴 도출 — 진짜 P/R/F1은 v2.9 후속 |
| 양식 정형성 측정 정량화 | 본 분석은 "HWP 평면 vs XLSX 정형" 이분 비교 — 정형성 점수 척도 정의 후속 필요 |

### 후속 (v2.9+)
- D1·D2·D4를 v2.8 19종으로 일관 재평가 → 4 데이터셋 완전 직접 비교표
- D5와 D1을 비교한 **공공 도메인 시점 차이 (2013 → 2024)** 분석
- "양식 정형성 지수" (form_formality_score) 정의 — S18·S15·S5 등의 가중 합
